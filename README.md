# **Image Chat**

一个基于 Flask 的 AI 图像处理工具。支持通过自然语言指令编辑图片，或通过文本描述生成新图片。

目前集成了 Google Gemini、OpenRouter 和 兔子 (Tuzi) 三种 API 服务商。

## **功能**

* **图像编辑**：上传图片（支持多张），输入指令（如“把背景改成星空”），AI 会自动处理。  
* **图像生成**：直接输入提示词生成图片。  
* **本地保存**：所有生成的图片会自动保存在 output 文件夹中。  
* **多模型切换**：可以在前端界面直接切换不同的 AI 模型和 API Key。

## **快速开始**

### **1\. 环境配置**

需要 Python 3.7+。

\# 创建虚拟环境  
python \-m venv venv

\# 激活虚拟环境 (Windows)  
venv\\Scripts\\activate

\# 安装依赖  
pip install \-r requirements.txt

### **2\. 运行**

python app.py

启动后访问：http://127.0.0.1:5000

## **使用说明**

1. 打开网页后，先在顶部选择服务商（如 Google），并输入对应的 API Key，点击保存。  
2. **编辑模式**：上传图片 \-\> 输入修改指令 \-\> 点击“开始编辑”。  
3. **生成模式**：输入画面描述 \-\> 点击“开始生成”。  
4. 生成的图片可以直接在网页预览或下载。

## **项目结构**

* app.py: 程序入口。  
* routes/: 路由逻辑（网页和 API 接口）。  
* services/: 核心逻辑（调用各个 AI 厂商的接口）。  
* templates/ & static/: 前端页面和样式。  
* output/: 存放生成结果的目录。
